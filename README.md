### Project Overview
This repository serves as a Digital Literacy Portfolio developed for the CSE0001 course. As a Student Digital Ambassador, my role is to assist peers in navigating the digital world—from professional profile creation to safe online communication. This project documents my work across five core modules, showcasing essential digital skills required for the modern academic and professional environment.

---

### Module Summaries

#### Module 1: Digital Literacy Awareness
* **Task:** Create a Digital Literacy Awareness Infographic.
* **Summary:** Developed a visual resource to help batchmates understand the definition of digital literacy and its importance. The infographic covers core topics such as essential digital tools and safe internet practices.

#### Module 2: Student Digital Portfolio
* **Task:** Build Your Student Digital Portfolio.
* **Summary:** Established a professional digital presence by setting up and optimizing accounts on platforms like GitHub and LinkedIn. This includes a personalized GitHub profile README and updated educational details on LinkedIn.

#### Module 3: Coding & Collaboration Platforms
* **Task:** Explore Coding & Collaboration Platforms.
* **Summary:** Demonstrated technical proficiency by completing a beginner-level challenge on a coding platform and creating a digital literacy awareness quiz using Google Forms to facilitate peer-to-peer learning.

#### Module 4: Professional Communication
* **Task:** Draft a Professional Email & Etiquette Guide.
* **Summary:** Developed professional communication skills by drafting formal emails for academic and internship inquiries. Additionally, created a social media checklist to promote responsible digital citizenship.

#### Module 5: Cybercrime Awareness
* **Task:** Cybercrime Awareness Case Study & Prevention Guide.
* **Summary:** Conducted a case study on common digital threats (such as Phishing or UPI fraud) and compiled an actionable "Stay Safe Online" checklist to help students recognize and avoid online risks.
